Print Design Case Study

Sanzo Wada
Color utility

Translating a 1930s physical masterpiece into a high-performance digital utility for modern print workflows.

Role Web Development
Archival
Timeline 4 Weeks
2024
Tools Html, Css
Vanilla Js
Context Self-Initiated

01. The Challenge

Browsing hundreds of color combinations in physical print is a beautifully tactile experience, but highly impractical for modern digital workflows. Translating these historical swatches into HEX equivalents manually and testing them creates a tedious bottleneck for print designers.

02. Swiss Block Wheel

To elevate the utility beyond a simple gallery, I engineered an interactive color wheel inspired by the grid-based layout of Swiss International Style. This block-style interface allows users to dynamically filter and select functional color combinations with modernist precision.

03. Execution

I developed a lightweight web application tailored for rapid extraction. The interface relies entirely on vanilla JavaScript and CSS to render hundreds of interactive color nodes seamlessly.

Layout Architecture

The layout respects the breathing room required for accurate visual contrast judgment. The grid framework directly mirrors the spatial rhythm of the original 1930s typesetting.

Micro-Interactions

Tactile feedback was integrated directly through CSS and simple JS. Copying a HEX code triggers a fluid tooltip and subtle color pulse, assuring the user without disrupting their workflow.
